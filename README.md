## Project

<b>Problem:</b><br>
<i>InvisoPollutant Tracker</i><br>
Most people spend 90% of their time indoors, unaware of VOCs or stagnant CO2 levels. Create a holistic "Home Environment Dashboard" that correlates air quality spikes with physical health symptoms.

<b>Solution:</b><br>
Our proposed solution is an integrated air quality monitoring system comprising both hardware and software components. The system captures live data on CO₂, VOC, PM1, PM2.5, and PM10 levels using dedicated sensors, and presents this information through a graphical user interface (GUI) that displays real-time readings, correlates them with potential health symptoms, and activates an audible buzzer alert when gas concentrations exceed permissible thresholds.
